## Naming Notes

- `Macchiato` is the APK icon name used for latte macchiato.
- `Creme` is treated as caffe crema / creme coffee.
- `Water` is represented as a hot-water beverage icon for machine-menu use.
- `MyCoffee` is represented as a neutral custom-recipe drink icon.
- `Undefined` is represented as a neutral placeholder badge for unknown or unmapped icon ids.
